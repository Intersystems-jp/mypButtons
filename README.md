# mypButtons
mypButtons routine creates one CSV (mgstat and Windows Performance are shown in one line) from multiple pButtons. This routine is for Windows.

# Background
InterSystems IRIS family has a nice utility **^SystemPerformance** (it's called as **^pButtons** in Cach√© and Ensemble) which outputs the database performance into a readable HTML file. When you run ^SystemPerformance on IRIS for Windows, a HTML file is created, and our specific performance log "mgstat" and Windows Performance log are included there. Test2
